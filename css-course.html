Css Course Concepts

CSS selectors are patterns used to select and style HTML elements. They include:

  • Element Selector: targets all elements of a specific type, e.g., p.
  • Class Selector: targets elements with a specific class, e.g., .menu.
  • ID Selector: targets a single element with a specific ID, e.g., #header.
p { color: blue; }
.menu { font-size: 18px; }
#header { background-color: lightgray; }

The CSS Box Model describes how elements are rendered. Each box consists of:

  • Content: The actual content of the box, such as text or an image.
  • Padding: Clears space around the content, inside of the border.
  • Border: A border surrounding the padding (if any) and content.
  • Margin: Clears space outside the border.
div {
  margin: 20px;
  border: 1px solid black;
  padding: 10px;
}

Flexbox makes it easier to design flexible and responsive layouts.

  • display: flex enables flex container behavior.
  • justify-content aligns items horizontally.
  • align-items aligns items vertically.
nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

CSS Grid provides a two-dimensional layout system for web design.

  • Create rows and columns using grid-template-rows and grid-template-columns.
  • Place items using line numbers or named areas.
.container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 10px;
}

Media queries help make websites responsive across different devices.

  • They apply different styles based on screen size, orientation, and resolution.
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

CSS positioning lets you control how elements are placed in the document flow.

  • static: Default position.
  • relative: Positioned relative to itself.
  • absolute: Positioned relative to nearest positioned ancestor.
  • fixed: Positioned relative to the viewport.
  • sticky: Scrolls with the page, then sticks at a position.
.box {
  position: absolute;
  top: 10px;
  left: 20px;
}

The z-index property controls the vertical stacking order of elements.

.modal {
  position: absolute;
  z-index: 1000;
}

Pseudo-classes define special states of elements. Common ones include:

  • :hover – when the mouse is over an element
  • :nth-child(n) – selects the nth child
a:hover {
  color: red;
}

li:nth-child(odd) {
  background: #f0f0f0;
}

Transitions enable smooth changes between property values over a given duration.

button {
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: #333;
}

CSS animations allow elements to move or change style over time.

@keyframes slidein {
  from { transform: translateX(-100%); }
  to { transform: translateX(0); }
}

.element {
  animation: slidein 0.5s ease-in-out;
}
