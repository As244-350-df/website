Html Course Concepts

HTML forms allow users to input data. Forms include elements like input fields, checkboxes, and buttons.

<form>
  <label>Name: <input type="text" name="name"></label>
  <button type="submit">Submit</button>
</form>

Semantic HTML introduces meaning to the web page. Tags like <header>, <article>, <section> provide better structure.

<article>
  <h2>Blog Post</h2>
  <p>This is a sample blog post.</p>
</article>

The <video> element allows embedding video files directly into web pages.

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>

Tables are used to display tabular data using <table>, <tr>, <td>, <th> elements.

<table>
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Alice</td><td>30</td></tr>
</table>

Anchor tags are used to link to other pages or sections on the same page.

<a href="https://example.com">Visit Example</a>

HTML supports ordered (<ol>) and unordered (<ul>) lists for organizing items.

<ul><li>Apple</li><li>Banana</li></ul>

Meta tags provide metadata like description and viewport settings in the <head> of the document.

<meta name="description" content="HTML tutorial">

HTML headings range from <h1> to <h6>, used to define the importance of text.

<h1>Main Heading</h1>
<h2>Subheading</h2>

<iframe> allows embedding another HTML page inside the current page.

<iframe src="page.html"></iframe>

The <img> element is used to embed images. Always include an alt attribute for accessibility.

<img src="image.jpg" alt="Description">
